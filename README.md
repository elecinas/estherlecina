# Esther Lecina - Portfolio Profesional

Portfolio personal que integra proyectos de **Desarrollo Frontend, Diseño Gráfico y Narrativa Visual**. Una experiencia interactiva con estética retro-tech/cyberpunk construida con arquitectura modular y carga dinámica de datos.

## 🚀 Tecnologías y Herramientas
- **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
- **Frameworks:** Bootstrap 4.0 (Base de rejilla y utilidades).
- **Datos:** Gestión dinámica mediante archivos JSON.
- **SEO & Accesibilidad:** Implementación de JSON-LD (Schema.org), Sitemaps y semántica HTML5.

## 📂 Estructura del Proyecto
- `/js`: Lógica de renderizado dinámico y filtrado de proyectos.
- `/css`: Estilos globales y componentes interactivos (Card flip, modales).
- `data.json`: Fuente de verdad de todos los proyectos del portfolio.
- `index.html`: Landing page y biografía.
- `web.html`, `grafica.html`, `art.html`: Secciones especializadas con filtrado dinámico.

## 🛠️ Instalación y Uso
Para visualizar el proyecto localmente:
1. Clona el repositorio: `git clone https://github.com/elecinas/tu-repo.git`
2. Abre `index.html` en tu navegador (se recomienda usar un servidor local como "Live Server" en VS Code para la carga correcta de archivos JSON).
